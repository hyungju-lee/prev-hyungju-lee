git remote show [remote] 명령은 모든 리모트 브랜치와 그 정보를 보여준다. 리모트 Refs가 있지만 보통은 리모트 트래킹 브랜치를 사용한다. $ git remote show origin * remote origin Fetch URL: https://github.com/hyungju-lee/hyungju-lee.git Push URL: https://github.com/hyungju-lee/hyungju-lee.git HEAD branch: master Remote branches: gh-pages tracked master tracked Local branch configured for 'git pull': master merges with remote master Local refs configured for 'git push': gh-pages pushes to gh-pages (up to date) master pushes to master (up to date)
