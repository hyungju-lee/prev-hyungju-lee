Sass 알아보기
Gulp에서 지원하는 Sass 모듈은 집필 중인 현재 두 가지로, gulp-sass와 gulp-ruby-sass 입니다.
Sass는 본디 Ruby 기반 환경에서 사용했지만, Ruby 기반 환경에서 개발하지 않는 사용자를 위해 C/C++ 버전의 Sass 라이브러리인 libsass(http://libsass.org)로 Ruby 기반 환경이 아닌 곳에서도 활용할 수 있게 되었습니다.
Node.js 기반 환경에서는 libsass 라이브러리를 계량한 node-sass(https://github.com/sass/node-sass) 모듈을 사용할 수 있습니다.

node-sass 모듈을 가져다 Gulp 환경에 맞춰 개발한 것이 gulp-sass(https://npmjs.com/package/gulp-sass) 모듈입니다.
반면, Ruby 환경에 의존하는 Gulp 모듈은 gulp-ruby-sass(https://npmjs.com/package/gulp-ruby-sass)입니다.
gulp-ruby-sass는 현재 사용안하는 모듈이라고 나옴 (이 책은 2015년 책)

Sass를 CSS로 변경하는 업무라는 점에서는 동일하지만, 각 모듈 간에는 뚜렷한 차이가 있ㅡㅅㅂ니다.
처리 속도면에서는 C/C++ 버전인 gulp-sass가 더 빠르지만,
기능 면에서는 Ruby 버전인 gulp-ruby-sass가 더 다양합니다. gulp-ruby-sass는 현재 사용안하는 모듈이라고 나옴 (이 책은 2015년 책)
반면, gulp-sass는 Node.js 환경에서도 잘 사용할 수 있지만, gulp-ruby-sass는 Ruby를 운영체제에 설치해야 사용할 수 있습니다.
Windows와 달리 Mac OS X은 기본적으로 Ruby가 설치되어 있어 별도로 설치하지 않아도 됩니다.
