NPM에 배포한 모듈 다운로드하여 테스트하기
NPM에 배포한 모듈을 다운로드하여 정상적으로 작동하는지 테스트합니다.
다음을 참고하여 Git Bash에서 cd 명령어로 프로젝트 루트 디렉터리로 이동합니다.
npm install combinejss 명령어를 입력하여 실행한 후 NPM에서 모듈 패키지를 다운로드 합니다.
모듈 패키지는 루트 디렉터리에 저장되는데, [node_modules] 디렉터리 안에 [combinejss] 모듈 패키지 디렉터리가 들어 있습니다.
모듈을 테스트하려고 moduleTest.js 파일을 생성합니다.
다음처럼 combinejss 모듈을 호출하고 앞서 배운 모듈 실행 코드를 입력합니다.
마지막으로 Git Bash에서 node moduleTest.js 명령어를 실행합니다.
모듈로 생성한 all.js 파일을 열면 압축, 병합된 코드를 확인할 수 있습니다.
