Sass 데이터 유형에는 어떤 것들이 있나요?
Sass 언어는 일곱 가지 유형의 데이터를 지원하며, 자바스크립트 데이터 유형과 매우 유사합니다.
각 데이터 유형을 정리하면 다음과 같습니다.
Sass 스크립트를 다양하게 활용하려면 다음 유형을 잘 알아 둡니다.
숫자(Number)
정수나 소수, 단위(px, em 등)가 붙은 숫자
예 : 10, 1.5, 21px
문자(String)
작은 따옴표나 큰 따옴표가 붙은 문자
예 : "images/icon.png", "Nanum Gothic", "Helvetica"
색상(Colors)
색상 정보에 해당되는 문자
예 : #fe4598, rgba(0,0,0,0.2)
불린(Boolean)
참 또는 거짓
예 : true, false
리스트(Lists of values)
자바스크립트 배열과 유사(공백이나 콤마로 구분하여 나열)
예 : 2px solid gray, Helvetica, Sans-serif
맵(Maps)
자바스크립트 객체와 유사
예 : (color:white, background-color:hotpink)
널(null)
'값이 비어 있음'을 의미하며, 값을 초기화할 때 사용
예 : null
