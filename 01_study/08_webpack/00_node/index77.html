EJS scripts 문법

EJS scripts 문법 - 조건문
조건문도 <% %> 안에 씁니다. 
                    
                        <% if (isLoggedIn) { %>
                            <div>로그인 되었습니다.</div>
                        <% } else { %>
                            <div>로그인이 필요합니다.</div>
                        <% } %>
case 문도 사용 가능합니다. 
                    
                        <% switch(fruit) { %>
                            <% case 'apple' : %>
                                <p>사과입니다.</p>
                            <% case 'banana' : %>
                                <p>바나나입니다.</p>
                            <% case 'orange' : %>
                                <p>오랜지입니다.</p>
                            <% default : %>
                                <p>사과도 바나나도 오랜지도 아닙니다.</p>
                        <% } %>
