Node.js에서 ES6 활용할 수 있는 환경 꾸미기
1. gulpfile.babel.js 생성 (기존엔 gulpfile.js로 생성했다면 .babel을 붙여줍니다.) 
                    
                        touch gulpfile.babel.js
2. Babel 설치 (ES2015 + 기능 및 호환성 문제를 피하기 위해 설치합니다.) 
                    
                        npm i -D @babel/register @babel/core @babel/preset-env
3. .babelrc 파일 생성 (babel 환경설정을 하는 json 파일입니다.) 
                    
                        touch .babelrc
4. .babelrc 파일을 열고 아래 내요을 입력합니다.
(babel-preset-env는 babel-preset-latest (또는 babel-preset-es2015, babel-preset-es2016 및 babel-preset-es2017과 동일)와 정확히 동일하게 작동합니다.) 
                    
                        {
                            "presets": [ "@babel/preset-env" ]
                        }
이렇게만 하시면 Node.js에서 Es6를 사용할 준비는 끝납니다.
